    CKAN Data API

    Access resource data via a web API with powerful query support. Further information in the main CKAN Data API and DataStore documentation.

    Endpoints

    The Data API can be accessed via the following actions of the CKAN action API.

    Query https://montreal.l3.ckan.io/api/3/action/datastore_search
    Query (via SQL) https://montreal.l3.ckan.io/api/3/action/datastore_search_sql

    Querying

    Query example (first 5 results)

    https://montreal.l3.ckan.io/api/3/action/datastore_search?resource_id={{resource.id}}&limit=5

    Query example (via SQL statement)

    https://montreal.l3.ckan.io/api/3/action/datastore_search_sql?sql=SELECT * from "{{resource.id}}" LIMIT 5

    Example: Javascript

    A simple ajax (JSONP) request to the data API using jQuery.

      
                        var data = {
                          resource_id: '{{resource.id}}', // the resource id
                          limit: 5 // get 5 results
                        };
                        $.ajax({
                          url: 'https://montreal.l3.ckan.io/api/3/action/datastore_search',
                          data: data,
                          dataType: 'jsonp',
                          success: function(data) {
                            alert('Total results found: ' + data.result.total)
                          }
                        });

    Example: Python

                          import urllib2
                      url = 'https://montreal.l3.ckan.io/api/3/action/datastore_search?resource_id={{resource.id}}&limit=5'
                      fileobj = urllib2.urlopen(url)
                      print fileobj.read()
    NOTE: This snippet only works with python 2.
