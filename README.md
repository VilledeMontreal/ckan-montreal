# CKAN NG EXAMPLE THEME

This is a placeholder for a CKAN exmaple theme.

## Installation

To install to your existing CKAN NG frontend:

```
$ yarn add ckan_ng_example_theme
```

in `.env`:
```
THEME=ckan_ng_example_theme
```
